# MAILER_API

This project is developed with spring boot in backend and vuejs in frontend.

# Features

- WebSocket with the spring implementation, <a href="https://stomp-js.github.io/">stompjs</a>.
- Send emails from the views to the smtp clients.
- Store temporally emails in redis.
- Vuex for manage the state for receive and send emails.
